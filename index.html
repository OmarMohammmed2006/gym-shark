📤 Sync Data
How to sync between phone and PC
1On your phone, tap Export Backup → saves a JSON file
2Transfer the file to your PC (WhatsApp, email, OneDrive, USB…)
3On your PC, tap Import Backup → pick the file → data merges

⚠️ Import merges data — it does NOT delete existing entries.
